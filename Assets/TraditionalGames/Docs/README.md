## 게임 구조 요약

### 영역 구성 (6개)
1. **Area_00_Loading** - 로딩 화면
2. **Area_01_Prologue** - 현대 경복궁 (시작)
3. **Area_02_Timeleap** - 타임리프 연출 + 도깨비 등장
4. **Area_03_Pandol** - Round 1: 판돌 건너기
5. **Area_04_Biseok** - Round 2: 비석치기
6. **Area_05_Ending** - 엔딩

### 게임 플로우
```
로딩 → 프롤로그 → 타임리프 → 판돌 건너기 → 비석치기 → 엔딩
```
