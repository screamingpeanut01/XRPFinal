# XRPFinal - 전통 게임 VR 프로젝트

## 🚀 시작하기

### 1. 프로젝트 Clone
```bash
git clone https://github.com/screamingpeanut01/XRPFinal.git
```

### 2. 대용량 에셋 다운로드 ⚠️

GitHub 용량 제한으로 일부 대용량 에셋은 별도로 다운로드해야 합니다.

| 에셋 | 용량 | 다운로드 링크 | 설치 경로 |
|------|------|--------------|----------|
| 경복궁 (GeunjeongjeonComplex) | ~4GB | [Google Drive 링크 추가 필요] | `Assets/GeunjeongjeonComplex/` |

**다운로드 후 해당 폴더에 압축 해제하세요!**

### 3. Unity로 프로젝트 열기
- Unity 버전: [Unity 버전 기입 필요]
- 프로젝트 폴더 열기
